# 多地 Ping 检测 - Chrome 扩展

## 功能特性

- 🔍 一键检测当前网站在全国多节点的访问延迟
- 🇨🇳 支持 120+ 全国主要城市节点
- 📊 可视化延迟地图和柱状图
- ⚡ 快速响应，结果实时展示
- 🎨 美观大气的 UI 设计

## 技术方案

由于浏览器无法直接发送 ICMP ping，使用 HTTP 请求测量响应时间：
- 对目标站点发起 HEAD/GET 请求
- 测量从请求到响应的时间差
- 通过不同地区的服务器 IP 模拟多地检测

## 使用方法

1. 打开 Chrome 扩展管理页面 `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `multi-ping` 文件夹
5. 点击插件图标即可使用
